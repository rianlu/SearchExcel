# SearchExcel
根据行列搜索Excel中的内容



![preview](https://github.com/rianlu/SearchExcel/blob/main/previewImages/preview.png)



![data](https://github.com/rianlu/SearchExcel/blob/main/previewImages/data.png)



注意事项：

如果直接使用 Maven 仓库的包：[poi-ooxm](https://mvnrepository.com/artifact/org.apache.poi/poi-ooxml/5.0.0)

会有以下冲突：

```bash
Duplicate class org.apache.batik.Version found in modules...
... ...
```

所以用了别人精简过的 `jar` 包



> [Android 读取excel （支持 xls和xlsx）](https://blog.csdn.net/u014743890/article/details/91045033)
>
> [android5xlsx](https://github.com/andruhon/android5xlsx)
